# Electron App Icons

Place your app icons here before running `npm run build`:

| File | Platform | Size |
|------|----------|------|
| `icon.ico` | Windows | 256×256+ (multi-size ICO) |
| `icon.icns` | macOS | 512×512 (ICNS bundle) |
| `icon.png` | Linux / tray | 512×512 PNG |

> The tray icon is resized to 16×16 automatically at runtime.
> For best results, use a square image with transparent background.
